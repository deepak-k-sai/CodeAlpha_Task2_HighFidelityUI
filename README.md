# CodeAlpha Task 2 - High Fidelity UI Design

## Project Name
Foodie - Food Delivery Mobile App UI

## Description
This project is a high-fidelity UI design created for a food delivery mobile application. The design was created in Figma by converting a low-fidelity wireframe into a colorful and user-friendly interface.

## Screens Included
1. Welcome Screen
2. Login Screen
3. Home Screen
4. Food Details Screen
5. Cart Screen

## Tools Used
- Figma
- UI/UX Design Principles

## Design Features
- Clean mobile app layout
- Orange color palette
- Food category cards
- Restaurant listing section
- Food details page
- Cart and checkout screen
- Rounded buttons and cards
- Simple typography

## Color Palette
- Primary Color: #FF6B35
- Background Color: #FFF8F2
- Text Color: #222222
- Grey Text: #777777
